IMPORTANT NOTES
The module system is designed to be modular and expandable. Each module file contains comprehensive standards for that specific research type. The CORE file contains universal standards that apply to all papers. When modules are loaded together, they provide targeted, relevant feedback instead of overwhelming users with standards that don't apply to their research type.
The Writing Workflow is intentionally separate from manuscript analysis - it's a resource for the writing process itself, not for analyzing completed manuscripts.
Users should be able to easily understand which type their paper is, and the help system should guide them if uncertain. The auto-detect feature with confirmation provides a safety net for confused users while still giving them final control.
